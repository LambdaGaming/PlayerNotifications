Simple addon that replaces the default player join/leave notifications with nicer looking ones. Also available on the [Steam Workshop.](https://steamcommunity.com/sharedfiles/filedetails/?id=3748867675) All features are listed below:
- Notifications will display when a player connects, fully loads in, and disconnects
- Player names are colored based on the player's team
- Admins will be able to see the Steam ID of each player that loads in and disconnects
- Notifications will not play any sounds
